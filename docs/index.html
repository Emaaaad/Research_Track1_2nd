Advanced Robotics Navigation Assignment

This documentation covers the ROS package developed for the Research Track I - Second Assignment by Seyed Emad Razavi. The package manages a robot’s movement within a Gazebo simulation, encapsulating three main ROS nodes for navigation, target retrieval, and data processing.
