Installation Guide

Prerequisites

  • ROS (Recommended Version: Noetic)

  • Python 3.x

  • Relevant ROS packages: geometry_msgs, nav_msgs, sensor_msgs, tf, actionlib, std_srvs

Clone the Repository

To install the project:

cd ~/catkin_ws/src
git clone https://github.com/Emaaaad/Research_Track1_2nd.git

Ensure Executable Permissions

chmod +x ~/catkin_ws/src/Research_Track1_2nd/scripts/*.py
